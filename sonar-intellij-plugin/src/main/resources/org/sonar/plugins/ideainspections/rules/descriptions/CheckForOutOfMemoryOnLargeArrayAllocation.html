This inspection is intended for J2ME and other highly resource constrained environments. Applying the results of this inspection without consideration might have negative effects on code clarity and design.

This inspection reports any instances of large array allocations which do not check for java.lang.OutOfMemoryError. In memory constrained environments, allocations of large data objects should probably be checked for memory depletion.
