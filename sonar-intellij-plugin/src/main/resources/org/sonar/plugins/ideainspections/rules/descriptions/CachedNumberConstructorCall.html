This inspection reports any attempt to instantiate a new Long, Integer, Short or Byte object from a primitive long, integer, short or byte argument. It may be more efficient to use the static method valueOf() here (introduced in JDK 5.0), which will cache objects for values between -128 and 127 inclusive. This inspection ignores calls to number constructors using a String argument.

For safety, this inspection is disabled unless the project uses JDK 5.0.
